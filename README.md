  # Situação-Problema Escolhida
  
Circuito Saquarema Verde
O projeto “Saquarema Verde Online” foi desenvolvido com o objetivo de facilitar o acesso às informações sobre ecoturismo, biodiversidade, trilhas, cachoeiras e eventos ambientais da cidade de Saquarema.
A plataforma busca centralizar informações importantes sobre:
•	Parque Estadual da Costa do Sol
•	Reserva Ecológica Estadual de Jacarepiá
•	Trilhas ecológicas
•	Eventos ambientais
•	Regras de visitação
•	Dicas de segurança
•	Horários de funcionamento
O sistema também promove conscientização ambiental e turismo sustentável.

# Descrição do MVP
O MVP (Minimum Viable Product) consiste em um website responsivo que permitirá aos visitantes acessarem rapidamente informações atualizadas sobre atrações naturais de Saquarema.
O sistema possui:
•	Página inicial com destaques ecológicos
•	Área de trilhas e eventos
•	Sistema de login para administradores
•	Painel administrativo simplificado
•	Interface intuitiva e responsiva
•	Layout moderno voltado ao ecoturismo

# Fluxo do Usuário
1.	Usuário acessa a página inicial.
2.	Navega pelas atrações.
3.	Consulta trilhas e eventos.
4.	Visualiza horários e dicas.
5.	Administrador acessa login.
6.	Administrador atualiza informações.

# Melhorias Futuras
•	Integração com banco de dados
•	Sistema de cadastro de usuários
•	API meteorológica
•	Geolocalização de trilhas
•	Sistema de avaliações
•	Chat para suporte turístico

# Conclusão
O MVP “Saquarema Verde Online” oferece uma solução moderna, sustentável e acessível para facilitar o acesso às informações ecológicas e turísticas da cidade.
A plataforma promove conscientização ambiental, incentiva o ecoturismo e melhora a experiência dos visitantes por meio de uma interface intuitiva e responsiva.
Além de atender aos requisitos propostos no desafio acadêmico, o projeto também possui potencial para expansão futura e aplicação real no município.
