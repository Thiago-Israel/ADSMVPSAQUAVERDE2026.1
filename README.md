# Situação-Problema: Circuito Saquarema Verde

O projeto surge da necessidade de um canal centralizado e confiável que reúna informações atualizadas sobre o ecoturismo em Saquarema, especialmente sobre o Parque Estadual da Costa do Sol e a Reserva Ecológica Estadual de Jacarepiá. A proposta é facilitar o acesso a dados relevantes para visitantes, moradores e administradores da região.

Resumo das Tecnologias Utilizadas

Para o desenvolvimento do MVP Front-End, foram empregadas as seguintes ferramentas e tecnologias:

HTML: Estruturação das páginas.

CSS: Estilos visuais e organização do layout.

JavaScript: Implementação de interações e funcionalidades dinâmicas.

Figma: Desenvolvimento dos protótipos das interfaces.

GitHub: Versionamento do código e hospedagem do repositório.

# Como Executar o MVP
Pré-requisitos

Node.js

Git

Yarn (opcional)

Passos para Rodar Localmente

Clonar o repositório

git clone https://github.com/


Acessar a pasta do projeto

cd ADs


Instalar as dependências

Com Yarn:

yarn install


Com NPM:

npm install


Iniciar o projeto

Com Yarn:

yarn dev


Com NPM:

npm run dev
