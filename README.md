  # Situação-Problema Escolhida
  
Circuito Saquarema Verde
O projeto “Saquarema Verde Online” foi desenvolvido com o objetivo de facilitar o acesso às informações sobre ecoturismo, biodiversidade, trilhas, cachoeiras e eventos ambientais da cidade de Saquarema.
A plataforma busca centralizar informações importantes sobre:
•	Parque Estadual da Costa do Sol
•	Reserva Ecológica Estadual de Jacarepiá
•	Trilhas ecológicas
•	Eventos ambientais
•	Regras de visitação
•	Dicas de segurança
•	Horários de funcionamento
O sistema também promove conscientização ambiental e turismo sustentável.

# Descrição do MVP
O MVP (Minimum Viable Product) consiste em um website responsivo que permitirá aos visitantes acessarem rapidamente informações atualizadas sobre atrações naturais de Saquarema.
O sistema possui:
•	Página inicial com destaques ecológicos
•	Área de trilhas e eventos
•	Sistema de login para administradores
•	Painel administrativo simplificado
•	Interface intuitiva e responsiva
•	Layout moderno voltado ao ecoturismo
